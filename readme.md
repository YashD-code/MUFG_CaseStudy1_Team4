# Excel Frontend (Task 1) - Streamlit UI

## Run locally
1. Create & activate venv
2. pip install -r requirements.txt
3. streamlit run app.py

## What this UI does
- Home (app.py)
- Upload (pages/2_File_Upload)
- Transformation input forms (pages/3_Transformation_Selection)
- Frontend-only preview (pages/4_Preview)
- Download preview as Excel/CSV (pages/5_Output)

Note: This project implements Task 1 (Frontend UI/UX) only.
